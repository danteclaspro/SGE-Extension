  SGE-Extension

The tools included in this extension enhance the way the School Management System delivers its functionality, including layout improvements for better data visualisation, automatic form completion, generation of additional reports, simplified navigation between modules, and other usability enhancements. These features help reduce the time previously spent on such tasks by staff working in school administrative offices.

The School Management System (SGE) was initially released in 2005, serving as the academic control system for the Municipal Education Network of the city of Belo Horizonte, MG – Brazil. Although it has been updated over time, some functional requirements raised by users working directly within school units have not yet been addressed.

To better understand these needs, research and on-site observation of daily operations within these departments were conducted. As a result, the following functionalities were validated:

#### Features - Version 1.7.1 (11/11/2025)

- Generation of absence reports including student justifications  
  *(Attendance Module > Absence Justification Register)*
- Improved layout and class navigation on the daily absence register screen  
  *(Attendance Module > Daily Absence Register)*
- Generation of attendance summary listings including dates, highlighting bars, automatic exclusion of withdrawn students, and total absence count fields  
  *(School Records Forms Module > Attendance Summary Listing)*
- Generation of student assessment forms with automatic completion according to each student's status  
  *(Report Card Module > School Report Card Records and Reports > Reports > Report Card)*
- Navigation to the student management history from the movement history screen  
  *(Movement Module > Movement History)*
- Shortcuts to the most frequently used modules from the student management screen  
  *(Main Menu > Student Management Module)*
- Search by guardian/responsible party details with navigation to movement history  
  *(Movement Module > Search by Guardian)*
- Generation of control reports to support pedagogical record-keeping  
  *(School Records Forms Module > Forms by Class)*

#### Release Notes of the current version
- Added support for the Full-Time attendance summary format  
  *(two sessions per school day)*
- New list navigation option in the daily absence register
- Improved formatting of the monthly class consolidation report to facilitate the identification of transferred or reassigned students
- Warning message added during the student withdrawal process for students with extended-time accommodations registered
